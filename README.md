============================================================
Introduction
------------------------------------------------------------
    An Arduino library for display multi - function histogram on screen. 
Usage
------------------------------------------------------------
    1.Git clone this repository to your Arduino IDE's libraries directory.
    2.Run the demo about histogram on examples directory.
